# 前端栈索引（React 库 · Vue 框架 · 元框架）

> **叫法**：**React** 是 **UI 库**；**Vue** 官方定位为**渐进式框架**；**Next / Nuxt / Remix** 等是在二者之上的**元框架**（带路由、构建、SSR 等约定）。

## 快速选型（一句话）

| 倾向 | 常见选择 |
|------|----------|
| 要 **React 库** + 全栈/SSR 约定 | **Next.js** 或 **Remix**（元框架） |
| 只要 React 做 SPA、自己拼路由 | **Vite** + React + **React Router** |
| 要 **Vue 框架** + 官方路由/状态 | **Vue** + **Vue Router** + **Pinia** |
| Vue + 文件路由 + SSR 一体 | **Nuxt** |
| 纯 SPA、极快 DX | **Vite** + React 或 Vue |
