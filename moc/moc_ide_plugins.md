## 已有线索 / 链接

### Cursor · 主题

- **Familiar Java Themes**（`zerodind.familiar-java-themes`）  
  - **用途**：在 **Cursor** 里换一套接近 **IntelliJ IDEA / Eclipse** 习惯的配色，改的是 **Color Theme**（颜色主题，不是 Java 语言支持）。  
  - **安装**：`Extensions` 搜索 *Familiar Java Themes* → Install（Cursor 兼容 Open VSX / Marketplace 与 VS Code 类似）。  
  - **切换主题**：命令面板 **「Preferences: Color Theme」**（macOS 常用 **`Cmd+K` `Cmd+T`**）→ 选如 *IntelliJ Light Classic*、*Darcula* 等变体。  
  - **Marketplace**：https://marketplace.visualstudio.com/items?itemName=zerodind.familiar-java-themes
